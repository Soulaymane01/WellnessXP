## 📖 Overview
**WellnessXP** is a web application designed to educate young people about **sexual health** in an engaging, modern, and interactive way.

We transform learning into a **game-like experience** using:
- 🎥 Short video courses (30–60 s reels)
- 📖 Interactive stories with quizzes
- 🧩 XP points and level progression
- 🎁 Real-life rewards (discounts, coupons)
- 🤝 A mentorship ecosystem where learners become educators

Our mission is simple yet powerful:
> *Don’t just teach one person — empower that person to educate their community.*

---

## 🚀 Features

| Feature | Description |
|----------|-------------|
| 🎬 **Mini-Courses** | Learn through short video reels (30–60 s) covering essential sexual health topics. |
| 🧠 **Gamification System** | Earn XP points, unlock levels, and track progress while learning. |
| 📖 **Interactive Stories** | Learn through stories that include quizzes and decisions to reinforce knowledge. |
| 🎁 **Rewards & Coupons** | Redeem XP for real-life benefits like discounts or coupons. |
| 🤝 **Mentorship Program** | Graduate learners become mentors and teach through partner associations. |
| 👤 **User Dashboard** | Track XP, completed lessons, rewards, and achievements. |

## 💡 How It Works

1. **Learn:**  
   Watch 30–60 s video lessons and explore interactive stories.

2. **Play:**  
   Earn XP by completing quizzes, stories, and challenges.

3. **Earn:**  
   Redeem XP for real-world rewards and coupons.

4. **Empower:**  
   Become a mentor and teach others through partner associations.
